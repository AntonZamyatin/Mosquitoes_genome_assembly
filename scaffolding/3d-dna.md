### run Juicer
```

module load bwa 
module load jdk

./scripts/juicer.sh -S "early" -g arabiensis -s Arima \
-z arabiensis.fasta -y arabiensis_Arima.txt -p assembly \
-q defq -l defq -D [juicer directory] 
```
